# Basic PathTracer
A very basic path tracer written in c++

# Features
- Support for diffuse and specular shading
- Support for hdr
- Result is rendered as a bimap

# Build Instructions
```
cmake --build ./build

./build/RayTracing
```

# Dependencies
[stb_image](https://github.com/nothings/stb/blob/master/stb_image.h)
